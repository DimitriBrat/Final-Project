About This Project

This is a web page created for INST377. It allows the user to search a song and/or artist and receive recommendations on similar songs/artists. The user can then add these recommendations to a database and access their database via the "User Information" tab on the home page.

Due to recency limitations, some songs will not have results. Older songs (pre-2005) by popular artists work best.

Technologies used: HTML, CSS, JavaScript, Last.fm API, Supabase API, Node.js, Vercel Hosting
